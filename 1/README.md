# จงหาเส้นทางที่มีค่ามากที่สุด

### 1. รันโปรแกรม
1. รันคำสั่ง `go run` เพื่อรันไฟล์ `main.go`
```bash
go run main.go
```

### 2. Output ที่ได้
```plaintext
ผลรวมเส้นทางที่มากที่สุด: 7273
```

## หมายเหตุสามารถนำค่าของพีระมิดไปใส่ไว้ใน input.json ได้เช่น 
```json
[
  [2],
  [3, 4],
  [6, 5, 7],
  [4, 1, 8, 3]
]
```
ผลลัพธ์
```plaintext
ผลรวมเส้นทางที่มากที่สุด: 21
```

# จับฉันให้ได้สิ ซ้าย-ขวา-เท่ากับ

### 1. รันโปรแกรม
1. รันคำสั่ง `go run` เพื่อรันไฟล์ `main.go`
```bash
go run main.go
```

### 2. ใส่ Input ตามที่โจทย์ให้มา หรือใส่อะไรก็ได้ที่เป็นเฉพาะ L R หรือ = เท่านั้น
```plaintext
input = LLRR=
input = ==RLL
input = =LLRR
input = RRL=R
```

### 3. ได้ Output ออกมาตามโจทย์ตามลำดับ

```plaintext
output = 210122
output = 000210
output = 221012
output = 012001
```
